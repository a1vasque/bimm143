Q11. On average is chocolate candy higher or lower ranked than fruit candy?

-first find all chocolate candy -find their winpercent values - calculate the mean of these values

  • then do the same for fruity candy and compare On average, the fruit candy is ranked higher than the chocolate candy
chocolate.inds <- candy$chocolate == 1
chocolate.win <- candy[chocolate.inds, ]$winpercent
c <- mean(chocolate.win)
fruity.inds <- candy$fruity == 1
fruity.win <- candy[fruity.inds, ]$winpercent
f <- mean(fruity.win)
c > f
